# Desktop Story

An Instagram-style story viewer component for the web.

## Features

- **Carousel Navigation** — Smooth horizontal carousel with up to 7 visible story cards
- **Substory Support** — Multiple videos per story with automatic progression
- **Video Playback Controls** — Play/pause, mute/unmute, and progress tracking
- **Keyboard Navigation** — Arrow keys to navigate, Space/Enter to play/pause
